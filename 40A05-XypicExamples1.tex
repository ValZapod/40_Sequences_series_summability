This documentation serves to illustrate the use of the xypic package, by giving examples of actual code and the corresponding results.  Before proceeding to add graphs, it is important to remember to add 
\begin{quote}\begin{verbatim}\usepackage{xypic}\end{verbatim}\end{quote}
to the preamble first!

\subsection{Basic Examples}

\begin{enumerate}
\item

\begin{multicols}{2}

\begin{verbatim}
\begin{center}
$\xymatrix{a\ar[r] & b}$
\end{center}
\end{verbatim}

\columnbreak

\begin{center}
$\xymatrix{a\ar[r] & b}$
\end{center}
\end{multicols}
