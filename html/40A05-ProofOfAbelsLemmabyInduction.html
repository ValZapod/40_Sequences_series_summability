proof of Abel’s lemma (by induction)


Proof. The proof is by induction. However, let us first recall that sum on the right side is a piece-wise defined function of the upper limit N-1. In other words, if the upper limit is below the lower limit 0, the sum is identically set to zero. Otherwise, it is an ordinary sum. We therefore need to manually check the first two cases. For the trivial case N=0, both sides equal to a0⁢b0. Also, for N=1 (when the sum is a normal sum), it is easy to verify that both sides simplify to a0⁢b0+a1⁢b1. Then, for the induction step, suppose that the claim holds for some N≥1. For N+1, we then have

∑i=0N+1ai⁢bi = ∑i=0Nai⁢bi+aN+1⁢bN+1
= ∑i=0N-1Ai⁢(bi-bi+1)+AN⁢bN+aN+1⁢bN+1
= ∑i=0NAi⁢(bi-bi+1)-AN⁢(bN-bN+1)+AN⁢bN+aN+1⁢bN+1.

Since -AN⁢(bN-bN+1)+AN⁢bN+aN+1⁢bN+1=AN+1⁢bN+1, the claim follows. □.
