upcrossings and downcrossings


Inequalities involving the number of times at which a stochastic process passes upwards or downwards through a bounded interval play an important role in the theory of stochastic processes. Whether or not a sequence of real numbers converges to a limit can be expressed in terms of the finiteness of the number of upcrossings or downcrossings, leading to results such as the martingale convergence theorem and regularity of martingale sample paths. As the main applications are to stochastic processes, we suppose that (Xt)t∈𝕋 is a real-valued stochastic process with totally ordered time index set 𝕋, usually a subset of the real numbers. However, for the definitions here, it is enough to consider Xt to be a sequence of real numbers, and the dependence on any underlying probability space is suppressed.

For real numbers a<b, the number of upcrossings of X across the interval [a,b] is the maximum nonnegative integer n such that there exists times sk,tk∈𝕋 satisfying

s1<t1<s2<t2<⋯<sn<tn (1)

and for which Xsk<a<b<Xtk. The number of upcrossings is denoted by U⁢[a,b]. Note that this is either a nonnegative integer or is infinite. Similarly, the number of downcrossings, denoted by D⁢[a,b], is the maximum nonnegative integer n such that there are times sk,tk∈𝕋 satisfying (1) and such that Xsk>b>a>Xtk.

Note that between any two upcrossings there is a downcrossing. Given sk,tk satisfying (1) and Xsk<a<b<Xtk, we can set sk′=tk and tk′=sk+1 for k=1,…,n-1. Then, Xsk′>b>a>Xtk′ from which we conclude that D⁢[a,b]≥U⁢[a,b]-1. Similarly, we have U⁢[a,b]≥D⁢[a,b]-1. Hence, the number of upcrossings and the number of downcrossings of an interval differ by at most 1.

For a finite index set 𝕋={1,2,…,N}, the number of upcrossings can be computed as follows. Set t0=0 and define s1,s2,… and t1,t2,… by

tk=inf⁡{t∈𝕋:t≥sk,Xt>b}∈𝕋∪{∞}.
sk=inf⁡{t∈𝕋:t≥tk-1,Xt<a}∈𝕋∪{∞}.

The number of upcrossings of [a,b] is then equal to the maximum n such that tn<∞ (see Figure 1).
