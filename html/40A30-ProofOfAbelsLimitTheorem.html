proof of Abel’s limit theorem


Without loss of generality we may assume r=1, because otherwise we can set an′:=arn, so that ∑an′⁢xn has radius 1 and ∑a′ is convergent if and only if ∑an⁢rn is. We now have to show that the function f⁢(x) generated by ∑an⁢xn (with r=1)is continuous from below at x=1 if it is defined there. Let s:=∑an. We have to show that

limx→1-⁡f⁢(x)=s.

If |x|<1 we have:

s-f⁢(x) =∑n=0∞an-∑n=0∞an⁢xn
=∑n=0∞(1-xn)⁢an
=(1-x)⁢∑n=1∞(xn-1+xn-2+…+x+1)⁢an
=(1-x)⁢∑n=0∞(s-sn)⁢xn

with sn:=∑i=0nai. Now, since s-sn→0 as n→∞ we can choose an N for every ε>0 such that |s-sn|<ε2 for all m>N. So for every 0<x<1 we have:

|s-f⁢(x)| <(1-x)⁢∑n=0m|rn|⁢xn+ε2⁢(1-x)⁢∑n=m+1∞xn
<(1-x)⁢∑n=0m|rn|+ε2.

This is smaller than ε for all x<1 sufficiently close to 1, which proves

limx→r-⁡∑an⁢xn=∑an⁢rn=∑limx→r-⁡an⁢xn.
