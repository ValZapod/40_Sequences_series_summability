The Dirac Delta Function

Definition

The Dirac delta function can be defined as

δ⁢(t)≜limϵ→0⁡1ϵ⁢Π⁢(tϵ)

where Π⁢(t) is the pulse function. The delta function can also be defined as

δ⁢(t)⁢[f⁢(t)]≜limϵ→0⁡1ϵ⁢∫-∞+∞Π⁢(tϵ)⁢f⁢(t)⁢𝑑t

1. ⁢∫-∞∞f⁢(t)⁢δ⁢(t-a)⁢𝑑t=f⁢(a)

Proof:

∫-∞∞f⁢(t)⁢δ⁢(t-a)⁢𝑑t =limϵ→0⁡(∫-∞a-ϵf⁢(t)⁢δ⁢(t-a)⁢𝑑t+∫1-ϵa+ϵf⁢(t)⁢δ⁢(t-a)⁢𝑑t+∫a+ϵ∞f⁢(t)⁢δ⁢(t-a)⁢𝑑t)
=limϵ→0⁡(∫a-ϵa+ϵf⁢(t)⁢δ⁢(t-a)⁢𝑑t)
=f⁢(a)⁢limϵ→0⁡(∫a-ϵa+ϵδ⁢(t-a)⁢𝑑t)=f⁢(a)⁢(1)=f⁢(a)
2. ⁢∫-∞∞f⁢(t)⁢δ⁢(t)⁢𝑑t=f⁢(0)

Proof: Readily seen if we set a=0 in Property #1.

3. ⁢∫-∞∞f⁢(t)⁢δ⁢(a⁢t)⁢𝑑t=1a⁢f⁢(0)
